# Changelog

## Ver.0.1.3 - 2026-02-25
- UI デザインを全面リニューアル
  - ソリッドな高コントラスト配色に変更（半透明・ブラー廃止）
  - 演算子にバイオレット、`=` にゴールドのデュアルアクセント配色
  - ダーク/ライト両テーマを刷新
- ファビコンを電卓シルエットの新デザインに変更（SVG + 全サイズ PNG 再生成）
- theme-color / webmanifest のカラーを新パレットに統一

## Ver.0.1.2 - 2026-02-24
- ダーク/ライトテーマ切替を追加（選択状態を保存）
- モバイル向け横向き固定を追加（起動時に自動判定）
- 横向き固定が使えない環境ではエラーを出さず `縦` 表示で固定

## Ver.0.1.1 - 2026-02-24
- 通常モード（縦）と関数モード（横）のレイアウト切替を追加
- `()` ワンタップ入力を追加（開き/閉じ括弧を自動判定）
- ボタン間隔とフォントサイズを操作性優先で最適化
- 参照用画像フォルダー `参照/` を削除

## Ver.0.1.0
- 初期リリース
